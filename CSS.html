CSS Básico

  1. Enlazar Css con Html

    para enlazarlo hay que agregar una etiqueta "link" en el head del html

    Ejemplo: "< link rel="stylesheet" href="ruta de archivo css" >"

  2. Selectores (Básico)

    • Selector Etiqueta

      Se utiliza el nombre de la etiqueta, haciendo que todas las modificaciones que se hagan se van a aplicar en todos los elementos que coincidan con esa etiqueta

      Ejemplo: p{
      propiedad:valor;
      }

    • Selector mediante Clases

      Una clase es un atributo que se utiliza para asignar un nombre a un grupo de elementos, permitiendo aplicarles estilos específicos o identificarlos fácilmente.

      Ejemplo: < p class="definiciones-parrafos" >contendio< /p >

      En css:
      .definiciones-parrafos{
      propiedad:valor;
      }

    • Selector mediante ID

      Es un identificador único, es decir, solo lo puede llevar un elemento

      Ejemplo:

      En Html: < p id="parrafoEspecial" >contenido< /p >

      En Css:

      #parrafoEspecial{
      propiedad:valor;
      }

  3. Propiedades de Texto y Fuente

    • color

      modifica el color del texto, hay varias formas de indicar el color con varios sistemas

      Ejemplo:

      p{
      color:red;
      }

    • font-family

      Indica la fuente de texto o tipografía de texto

      se puede escribir el nombre de la tipografía tal cual, pero cuando la tipografia tiene mas de una palabra se debe usar comillas simples ''; tambien se pueden agregar mas de una, separandolas por comas; eso por si la primera no funciona

      Ejemplo:

      h1{
      font-family: sans-serif, 'arial';
      }

    • font-size

      Modifica el tamaño de la letra, se suele dar la medida en pixeles, pero tambien hay otras alternativas

      Ejemplo:

      p{
      font-size:15px;

    • font-weight

      Sirve para modificar el grosor del texto; conocido para darle el aspecto negrita a la letra

      Ejemplo:

      p{
      font-weight:15px;

      La tipografias suelen tener definidas sierto niveles de grosor en la tipografia; es decir en vez de colocar en pixeles cierto numero, lo definencon la tipigrafía en sí, los niveles parten de 100 hasta 900 subiendeo de cien en cien; tambien se puede definir manualmente.

      Mas particularidades: si se usa "bold", el valor es como de 700; si se usa bolder, hace que vaya un escalon mas allá del normal de bold

    • font-style

      Define el estilo de la letra; hay varias opciones:

      • normal: es el valor por defecto, de la tipografia
      • Italic: es un poco inclinada
      • oblique: Tambien inclina la letra pero un poco menos de italic, en algunos casos es lo mismo

    • text-align

      Define el alineado de la letra; puede ser hacia la derecha, izquierda, centrado, justificado; en la practica serían:

      • rigth
      • left
      • center
      • justify
      • start
      • end

    • text-decoration

      es una propiedad con la se puede aplicar elementos decorativos o complementarios al texto; como un subrayado, el efecto de tachado etc; los principales son:

      • underline
      • overline
      • line-through

      Se pueden combinar entre sí, tambien hey otros valores pero esos son los más usados

      Otra particularidad es que el valor por defecto se llama "none" y significa que no hay ninguna decoración aplicada

    • line-height

      Definie el interlineado, es decir, el espacio que hay entre linea y linea; se usan valores decimales; donde 1 es como la base y se puede ir a 1.1; 1.5 etc; tambien se pueden usar porcentajes pero es relativo

    • letter-spacing

      define el espacion que hay entre letra y letra; se puede definir con pixeles

      Ejemplo:

      p{
      leeter-spacing:5px;
      }

    • text-transform

      Es una propiedad que permite cambiar el texto de formas como pasar todo a minúsculas o mayúsculas; hacer que la primera letra de cada palabra sea mayuscula etc; los principales valores son:

      • none (default)
      • uppercase
      • lowercase
      • capitalize

  4. Tipografías Externa

    Hay dos formas de traer Tipografías externas:
    • Agregando tipografía en el Head

      Se busca la tipografia en Google Fonts y luego en una etiqueta "link" se enlaza la dirección de la tipografía

      Para usarla se modifica la propiedad font-family, ademas de las propiedades de font-weight para el grososr de la letra y la de font-style para aplicar algún estilo

    • Descargando La tipografía y configurandola

      Se descarga la tipografía en Google Fonts y se siguen los siguientes pasos:

      1. Se descarga la tipografia del Google Fonts
      2. Se extrae el archivo, para encontrar la carpeta "static"
      3. Crear un nuevo archivo CSS, el nombre puede ser "fonts"
      4. Se vincula en el Head del Html
      5. Se llama la estructura "@font-face" que sirve para configurar la tipografia; el nombre; se agrega la dirección del archivo en la carpeta static; y tambien se define el font-weight
      6. Ya por último se puede usar la tipografia con el nombre que se le asigno y con el font-weight usar las diferentes variantes entre ellas, con tal y se hallan configurado

      Ejemplo de la configuración:

      @font-face {
      font-family:"Monserrat" ;
      src: url(/tipografias/regular.ttf) format("truetype");
      font-weight: 400;
      }

      Para configurar mas, se repite la misma estructura; ya que dentro de la carpeta static suele haber variantes de la misma tipografía

  5. Box Model

    Concepto que explica o actua en la relación de cada elemento html, cada elemento se representa como un bloque, y hay varias propiedades

    • content

      Es el contenido de la etiqueta, como el texto o una imagen etc

    • padding

      Es como un relleno, relleno entre el contenido y el borde del elemento

    • border

      Es el borde del elemento,al cual se le puede modificar el grosor, el estilo, el tamaño, color y forma; rodea tanto el contenido y el padding

    • margin

      El margen representa el espacio entre el elemento con otros elementos, no hace parte del elemento, solo es un espacio que lo separa de otros elementos

    Vamos a conocer las propiedades con mas detalle

    • width & height

      Es el ancho y alto del elemento, si no lo tiene definico o el contenido no lo define, es como si no existiera el elemento; se puede modificar con las propiedades width y con height; modificando el tamaño mediante pixeles, porcentajes u otras medidas relativas u absolutas

    • Padding y Margin

      Padding: espacio entre el contenido y el borde de la página, es desde el borde interno hasta el contenido; el tamaño del contenido no cambia con esta propiedad; esta propiedad engloba las cuatro direcciones (arriba, derecha, abajo, inzquierda), por separado serian padding-top, padding-right, padding-bottom, padding-left.

      Tambien podemos modifcar cada una desde "padding" ejemplo:

      padding: 10px (top) 5px (right) 3px (bootom) 2px (left)

      Tambien se puede modificar top y bootom al mismo tiempo y rigth y left tambien

      Ejemplo:

      padding: 10px (top & bootom) 5px (right & left)

      Margin: sirve o define el espacio entre un elemento con los elementos que lo rodean; tambien funciona como el conjunto de todas las direcciones; margin-top, margin-right, margin-bootom, margin-left; con margin se puede modificar todos agregando los cuatro valores (así sea 0); o los verticales y horizontales agregando dos valores

      Un característica importante del margin es la superposicón de los margenes de un elemento con otro, en el eje vertical, es decir, en el margin top y bootom dependiendo de la posicion de cada elemento; si el elemento de arriba tiene un margin-bottom de 10px y el elemento de abajo tiene un margin-top de 10px; el margen o espacio entre los dos elementos no es de 20px; no se suman los margenes, se superponen, en este caso sería de 10px; si uno de los es mayor que el otro, el margen que se aplica es el mayor.

      Una forma de centrar elementos con margin: la forma es darle el valor "auto", funciona para centrar horizontalmente un elemento en el espacio que lo contiene

    • border

      Esta propiedad engloba varias propiedades; estas en conjunto definen el borde como tal; son:

      • border-width: se aplica con unidades de medida como pixeles
      • border-style: los valores pueden ser solid,
      • border-color: se puede aplicar el color en cualqier formato que lo permita

      border puede modificar las tres propiedades, se dan los valores en el orden (tamaño estilo color)

      Ejemplo: border: 3px green solid

      Tambien funciona para modifcar un solo lado del elemento; se debe usar "border", luego el lado (top,right, botoom, left); y luego se aplican los tres valores

      Ejemplo: border-top: 5px solid blue

      además hay otra propiedad que sirve para redondear los bordes: border-radius

      border-radius: redondea las esquinas dependiendo del valor asignado, se puede dar en unidades absolutas como pixeles o relativas como porcentajes; hay diferencias entre usar una u otra
